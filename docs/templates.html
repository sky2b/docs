Templates

Commands to execute Go language templates.

<template-execute>

Executes template

<all>
    <!-- read HTML templates in "templates" directory -->
    <file-read type="template" file="templates/*.html" to="templates" glob="true"/>
    <!--
        execute the template above. By default "page.html" template is executed,
        otherwise a template name is taken from "data.template" namespace path.
        A template data object is taken from "data" path.
        And the resulting page is stored in a "page.content" variable.
     -->
    <template-execute path="templates" to="page.content" name="page.html" name-path="data.template" data-path="data"/>
</all>
  • path a namespace path to a template
  • to a namespace path where a generated content is stored
  • data-path a namespace path to template data
  • name an optional name of the template to execute
  • name-path an optional name of the template to execute (when given is used instead of the name attribute)
