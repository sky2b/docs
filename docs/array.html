Array functions

Components to transform arrays

<array>

Map, filter, aggregate, sort arrays of objects

<!-- iterate over the list of our books -->
<each path="data">
    <!-- for each offer -->
    <array path="listing.offers">
        <!-- find price plus shipping fees -->
        <expression>
            total = price + shippingPrice
        </expression>
        <!-- our price is the minimum "total" value -->
        <aggregate>
            <min path="total" to="price"/>
        </aggregate>
    </array>
    <print path="price"/>
</each>
  • path a namespace path to array
  • filter

    Filters away elements for which any of boolean expressions the tag contains is not true. There could be many filters and multiple boolean expressions each on a separate line. The syntax of the expressions is compatible with the syntax of Go language. Additionally when expression doesn't return a boolean value we use Javascript conventions: zero, empty string, nil - are treated as a boolean false, all other values as true.

  • first

    Take only first n elements of the array.

    • n the number of array elements to take
  • sort

    Sort the array.

    • path a namespace path to the property along which the array is sorted
  • expression

    Expression which is executed for every element of the array. If index or scope attributes are present, they give the name of a property which will contain element position or the namespace (or scope) of the &lt;array&gt; component. These values are removed from elements after expression was calculated.

    • index a name for element position in the array
    • scope a name for the scope object
  • empty a container, which is executed when array is empty
  • aggregate

    Configures aggregate functions executed on the array of objects. Average, sum, etc are found along some axes (object properties). There could be multiple aggregate functions called along deferent axes (properties).

    • avg

      Calculate an average value.

      • path a namespace path of the property to average
      • to a namespace path for the operation result
    • sum

      Calculate the sum of elements.

      • path a namespace path of the property to sum
      • to a namespace path for the operation result
    • min

      Find minimum.

      • path a namespace path of the property of which we want to find minimum
      • to a namespace path for the operation result
    • max

      Find maximum.

      • path a namespace path of the property of which we want to find maximum
      • to a namespace path for the operation result
    • count

      A total number of elements in the array.

      • to a namespace path for the operation result
