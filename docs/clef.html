Scripting

Components to execute built in scripting language Clef. The goal is to provide means to evaluate simple mathematical expressions and control statements while making syntax similar to the syntax of a Google Go language. In contrast to GO, Clef is a dynamically-typed language. To simplify common scripting tasks, it also has "matches regular expression" operator "~" (and opposite "!~") and follows Javascript conventions when converting values to booleans (0, empty string and nil are treated as false and all other values as true).

<clef>

Execute scripts in a built-in scripting language - Clef. Scripts have access to all namespace variables. You may assign values to new properties and you can use common control statements. There is only one global scope, associated with the current namespace. 

<!-- iterate over "data" array  -->
<each path="data">
    <!-- conditionally change the value of "num" property -->
    <clef>
        if rating > 1000 && number < 4 {
            number += 2
        }
    </clef>
</each>
