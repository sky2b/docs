Emails

Components to send emails.

<spark-post>

Send emails using sparkpost.com transactional email service

<all>
    <json>{
        "to":"client@example.com",
        "body": "test message",
    }</json>
    <spark-post key="YOUR_API_KEY" sender="your@example.com" to="to" subject="sandbox" body="body"/>
</all>
  • key an API key
  • sender a message sender
  • to a namespace path of a mail recipient
  • subject a message subject
  • body a namespace path of a message body
