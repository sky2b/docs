Getting started

Read series of articles highlighting different aspects of AddGears functionality based on examples in “getting-started” directory.

  • Installation. Installation instructions.
  • Basic concepts. Configuration, components and execution namespace.
  • Makefiles. Makefiles, targets, configuration parameters and templates.
  • File operations. Read and write files of various types: CSV, JSON, YAML, HTML templates.
  • SQL database. Modify database schema, save and query data using SQL.
  • Web server. Serve static HTML files and create API endpoints.
  • User authentication. Manage user permissions using JWT tokens.
  • Harvester. Crawl and cache HTML pages (Amazon listings) for easy processing (AddPricer components are required).
  • Web scraper. Extract data from HTML page using CSS selectors and a set of filters (AddPricer components are required).
  • Reprice. Reprice your books based on information about offers on Amazon site (AddPricer components are required).
  • Control statements. Conditional components execution.
  • Scripting. Use a built in scripting language.
